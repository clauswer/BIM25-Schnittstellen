# Aufgabe 1

Schreiben Sie ein Programm, das die Metadaten der neuesten Dokumente des OPUS Repository der TH Wildau abruft und in eine Datei speichert. 
Es sollte parametrisierbar sein, Abruf der Daten der letzten Woche, des letzten Monats sowie ein Zeitraum von Startdatum bis Enddatum. 
Beachten Sie, dass Sie alle Datensätze abrufen, Stichwort „ resumptionToken“. 

OPUS OAI URL: https://opus4.kobv.de/opus4-th-wildau/oai 

Dokumentation: 
- https://www.openarchives.org/OAI/2.0/guidelines-harvester.htm
- https://www.openarchives.org/OAI/openarchivesprotocol.html#Idempotency
